Studio Ghibli API

This API will be for Studio Ghibli films. It will be able to search movie titles, directors, and genres. Users will be able to create a their own profile, change their user information and make a list of their favorite films.

Request URL HTTP Method Response
Return a list of ALL movies '/movies' GET Returns a JSON list of movies.
Return data about a movie by title '/movies/:Title' GET Returns a JSON data about a specific movie.
Return data about a director "/directors/:directorName" GET Returns a JSON data about the director.
Return data about a genre description by genre name. "genre/:genre" GET Returns a JSON data with a description of that genre.
Allow new users to register /users POST Will add new user to JSON object. Username: Password: Email: Birthday:xx-xx-xxxx
Allow users to update their user information "/users/:Username" PUT Returns message that user data was updated.
Allow users to add a movie to their list of favorites "/users/:Username/movies/:MovieID" PUT Returns message affirming movie was added successfully.
Allow users to remove a movie from their list of favorites "/users/:Username/movies/:MovieID" DELETE Returns message affirming movie was removed successfully.
Allow users to remove their profile /users/:Username DELETE Return a message affirming that the user was deleted.
